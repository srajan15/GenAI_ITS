# Web Scraping Tasks

---

### **Task 1: Fetch Website Title**

Write a Python program to:

- Take a website URL as input
- Fetch the webpage
- Display the **title of the webpage**

---

### **Task 2: Extract Paragraph Text**

Write a program to:

- Download a webpage
- Extract and display **all paragraph (`<p>`) text**

---

### **Task 3: Count Hyperlinks**

Write a program to:

- Fetch a webpage
- Count the total number of **hyperlinks (`<a>` tags)**
- Display the count

---

### **Task 4: Extract Headings**

Write a program to:

- Extract all headings (`h1`, `h2`, `h3`)
- Display them on the screen

---

### **Task 5: Save Paragraphs to File**

Write a program to:

- Extract all paragraph text from a webpage
- Save the extracted text into a file named `content.txt`

---

### **Task 6: Extract Links with Text**

Write a program to:

- Extract all hyperlinks
- Display **link text along with the URL**

---

### **Task 7: Clean Readable Text Extraction**

Write a program to:

- Remove `script`, `style`, and `noscript` tags
- Extract and display only **clean, readable text**

---

### **Task 8: Count Common HTML Tags**

Write a program to:

- Count how many times the following tags appear on a webpage:
  `p`, `a`, `img`, `h1`, `h2`
- Display the count for each tag

---

### **Task 9: Extract Image URLs**

Write a program to:

- Extract all image URLs (`<img src="">`)
- Save the URLs into a file named `images.txt`
