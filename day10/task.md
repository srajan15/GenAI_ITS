Level 1

Task 1: Greeting App
Create a Gradio app that takes a name as input and returns a greeting message.
Requirements:

- Use gr.Textbox for input
- Use gr.Textbox for output
- Add a title
- Disable flagging

Task 2: Square Calculator
Create an app that takes a number and returns its square.
Requirements:

- Use gr.Number for input
- Use gr.Number for output
- Add examples

Task 3: Add Two Numbers
Create an app that takes two numbers and returns their sum.
Requirements:

- Two gr.Number inputs
- One gr.Number output
- Add labels
